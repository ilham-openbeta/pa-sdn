Tentang Aplikasi

Aplikasi ini merupakan aplikasi yang digunakan untuk memantau throughput jaringan Software Defined Network (SDN). Untuk saat ini jaringan SDN berjalan pada simulasi GNS3 menggunakan OpenvSwitch (Openflow Switch), ONOS (SDN Controller), dan sFlow-RT (sflow collector). Berikut ini merupakan penjelasan beberapa menu aplikasi ini.


Menu Topologi

Menu ini memiliki dua fitur yaitu :
  • Visualisasi topologi

    Topologi jaringan ditampilkan secara otomatis dan diperbarui setiap ada perubahan perangkat yang terhubung. Semua perangkat baru yang terdeteksi akan terletak ditengah layar, untuk merubahnya pengguna dapat menggeser gambar perangkat tersebut kemudian mengklik tombol "Simpan Bentuk Topologi" pada kiri atas layar. Selain itu, warna link antar perangkat akan berubah secara otomatis setiap 10 detik sesuai throughput. Throughput disini dihitung dari traffic inbound + outbound pada interface salah satu perangkat yang saling terhubung.
  • Informasi perangkat

    Pengguna dapat mengklik gambar perangkat untuk menampilkan panel informasi perangkat pada sebelah kanan layar. Panel ini juga digunakan untuk merubah nama perangkat pada sisi aplikasi untuk memudahkan identifikasi perangkat. Pengguna dapat merubah nama perangkat dengan mengklik Ubah nama. Pada panel ini juga terdapat pintasan untuk melihat grafik pada perangkat.

    • Menu Grafik

    Menu ini digunakan untuk melihat grafik throughput perangkat yang dipilih. Grafik ini diperbarui secara otomatis setiap 10 detik. Untuk penggunaan lebih mudah, menu grafik ini dapat dibuka melalui menu Topologi dengan memilih gambar perangkat kemudian mengklik Lihat Grafik >> pada panel informasi perangkat.
    Cara Penggunaan grafik: klik dan tahan untuk memperbesar, klik ganda untuk memperkecil, tombol shift + klik untuk menggeser.


    Menu Laporan

    Menu ini digunakan untuk melihat laporan throughput setiap perangkat pada periode waktu tertentu. Pengguna hanya perlu memilih periode yang akan ditampilkan kemudian klik tombol "Go" untuk menampilkan laporannya di bawah layar. Laporan dapat dicetak dengan menekan tombol Ctrl+P pada keyboard atau mengklik tombol "Cetak Laporan" yang terletak di bawah laporan.


    Menu Traffic generator

    Menu ini hanya digunakan untuk testing aplikasi.
