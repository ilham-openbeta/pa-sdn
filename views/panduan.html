Menu Topologi

Pada menu ini dapat menampilkan topologi jaringan secara otomatis. Pengguna dapat mengklik link atau ikon perangkat untuk menampilkan informasi perangkat. Selain itu, warna link antar perangkat akan berubah secara otomatis setiap 10 detik sesuai throughput pada interface perangkat tersebut. Secara default, nama perangkat/label menggunakan datapath ID (DPID). Pengguna dapat merubah nama perangkat dengan mengklik perangkat lalu mengklik tulisan Ubah nama lalu isikan namanya. Untuk menyimpan perubahan nama perangkat dan koordinat perangkat pada gambar topologi, gunakan tombol "Simpan Bentuk Topologi" di kiri atas layar.


Menu Grafik

Menu ini digunakan untuk melihat grafik throughput perangkat yang akan diperbarui secara otomatis sesuai pengaturan interval sflow pada sisi server (default 10 detik). Pengguna dapat melihat grafik perangkat berdasarkan Datapath ID (DPID) yang dipilih. Untuk penggunaan lebih mudah, menu grafik ini dapat dibuka melalui menu Topologi dengan memilih ikon perangkat kemudian mengklik Lihat Grafik >> pada panel informasi perangkat.


Menu Laporan

Menu ini digunakan untuk melihat laporan throughput setiap perangkat pada periode waktu tertentu. Pengguna hanya perlu memilih periode yang akan ditampilkan kemudian klik tombol "Go" untuk menampilkan laporan. Laporan juga dapat dicetak dengan menekan tombol Ctrl+P pada keyboard atau mengklik tombol "Cetak Laporan" pada bagian bawah laporan.
